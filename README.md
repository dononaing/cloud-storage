# cloud-storage
Stack:  
    Java 21    
    Spring Boot    
    Spring Security    
    PostgreSQL (credentials storage)    
    MinIO (main s3 file storage)    
    Redis (session cache)    
    Docker, Docker compose    
    Gradle    
    React  
# to run locally
Requirements: Docker (https://www.docker.com/products/docker-desktop/)  

  
step 1: download https://github.com/dononaing/cloud-storage/blob/main/docker-compose.yml    
step 2: open cmd  
step 3: cd "path where you installed docker compose file"  
step 4: docker compose up -d  
step 5: check: http://localhost:8080/
